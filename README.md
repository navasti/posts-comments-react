## Posts Widget
This project is a simple, mobile first widget which allows you to read posts, comments and filter them by headline.

## Getting Started
Install node_modules folder
```
npm install
```
Run the application
```
npm start
```

## Used Technologies
* HTML
* Web API
* SCSS - Mobile First Styles
* JavaScript - React with hooks
